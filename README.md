# ZenTask

ZenTask là một ứng dụng quản lý công việc (Todo List) hiện đại.

## Trạng thái hiện tại
Dự án đã được cấu hình để kết nối trực tiếp với cơ sở dữ liệu trên cloud (Aiven MySQL).

## Cách sử dụng
1. Clone dự án về máy.
2. Đảm bảo cấu hình trong `config/database.php` đã đúng (sử dụng kết nối từ xa).
3. Đặt file `ca.pem` (chứng chỉ SSL) vào thư mục `config/` để đảm bảo kết nối bảo mật.
4. Chạy qua XAMPP hoặc Web Server tương tự.
